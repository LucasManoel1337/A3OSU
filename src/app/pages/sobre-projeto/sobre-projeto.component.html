Sobre o Projeto

Sistema Distribuidos

Missão

Este projeto foi criado com o objetivo de centralizar e analisar dados de desempenho competitivo em tempo real. Nossa meta é fornecer métricas precisas, facilitar a gestão de torneios e oferecer uma interface intuitiva para que jogadores foquem no que realmente importa: a superação de seus limites.

Pilares Técnicos

Frontend Angular & CSS3 Interface reativa e componentes modulares.
Backend Java & Spring Boot API de alta performance para processamento.
Banco de Dados MySQL Consistência e integridade nos dados.
